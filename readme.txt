Modulates advertiser integration plugin. Plugin will automatically place the Modulates merchant integration code in your website. Works with WP-Ecommerce, WooCommerce and Cart66.

== Description ==

The Modulates Advertiser Integration helps advertiser to embed Modulates integration API into the WordPress based websites. If you are using WP-Ecommerce, WooCommerce or Cart66, you can use this plugin on your website. Simply sign up at http://modulat.es/wplg for your advertiser's account.

Generate huge traffic and sales to your website/business online using Modulates integration system and its Video Commerce Platform

The plugin doesn't come with any UI. So please do not expect any settings or admin page for it.

For support and feedback, please [click here](http://www.modulates.com/contact-us.php "Contact Modulates"). 

Click here for more information about [Modulates Video Commerce](http://www.modulates.com/how-it-works_video_affiliates.php "Modulates Video Commerce"). 

== Installation ==

Super easy installation.

1. Upload `modulates_advertiser_integration` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done!

== Frequently Asked Questions ==

= Where do I see settings? =

The plugin is so simple to use, it does not have any settings UI.

= How do I enter my Advertiser's username or account ID =

The plugin will auto recognize your advertiser's information once a publisher sends a visitor to your website.
